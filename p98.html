\[ \int\limits_0^1 \sqrt{1-x^2} dx = \int\limits_0^{\pi/2} \sqrt{1-\sin^2 t} \cos t dt = \int\limits_0^{\pi/2} |\cos t| \cos t dt = \int\limits_0^{\pi/2} \cos^2 t dt \]

\[ \frac{1}{2} \int\limits_0^{\pi/2} (\cos 2t + 1) dt = \frac{1}{2} \left( \frac{1}{2} \sin 2t + t \right) \Big|_0^{\pi/2} = \frac{1}{2} \left( \frac{1}{2} \sin \pi + \frac{\pi}{2} \right) = \frac{1}{2} \left( 0 + \frac{\pi}{2} \right) = \frac{\pi}{4} \]

–ü—Ä–∏–∫–ª–∞–¥ 16.5. –û–±—á–∏—Å–ª–∏—Ç–∏ —ñ–Ω—Ç–µ–≥—Ä–∞–ª \(\displaystyle \int\limits_0^{\pi/2} \frac{\cos x dx}{1+\sin^2 x}\)

–í–≤–µ–¥–µ–º–æ –∑–∞–º—ñ–Ω—É \(t = \sin x\). –¢–æ–¥—ñ \(dt = \cos x dx\). –û–±—á–∏—Å–ª–∏–º–æ –Ω–æ–≤—ñ –º–µ–∂—ñ —ñ–Ω—Ç–µ–≥—Ä—É–≤–∞–Ω–Ω—è:
