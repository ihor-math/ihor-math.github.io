Відстань від точки до прямої

Нехай пряма \( l \) задана своїм загальним рівнянням:

\[ Ax + By + C = 0, \]

а точка \( M_0(x_0; y_0) \) — довільна точка площини.

Відстань \( d \) від точки \( M_0 \) до прямої \( l \) визначається як модуль проекції вектора \( \overrightarrow{M_1 M_0} \), де \( M_1(x_1; y_1) \) — довільна точка на прямій \( l \), на нормальний вектор \( \vec{n} = (A; B) \) прямої (рис. 9.30).

Таким чином:

\[ d = |\operatorname{пр}_{\vec{n}} \overrightarrow{M_1 M_0}| = \frac{|\overrightarrow{M_1 M_0} \cdot \vec{n}|}{|\vec{n}|}. \]

Оскільки \( \overrightarrow{M_1 M_0} = (x_0 - x_1; y_0 - y_1) \) і \( \vec{n} = (A; B) \), то

\[ d = \frac{|(x_0 - x_1)A + (y_0 - y_1)B|}{\sqrt{A^2 + B^2}}. \]

Розкриваємо дужки:

\[ d = \frac{|Ax_0 + By_0 - Ax_1 - By_1|}{\sqrt{A^2 + B^2}}. \]

Точка \( M_1 \) лежить на прямій, тому \( Ax_1 + By_1 + C = 0 \implies C = -Ax_1 - By_1 \).

Підставляємо:

\[ d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}} \quad (9.20) \]

Формула (9.20) — відстань від точки \( M_0(x_0; y_0) \) до прямої \( Ax + By + C = 0 \).

Приклад 9.19. Знайти відстань від точки \( M(5; 4) \) до прямої \( 4x + 3y - 12 = 0 \).

Використаємо формулу (9.20):

\[ d = \frac{|4 \cdot 5 + 3 \cdot 4 - 12|}{\sqrt{4^2 + 3^2}} = \frac{|20 - 12|}{5} = \frac{8}{5} = 4. \]

Відповідь: \( d = 4 \).

Приклад 9.20. Знайти відстань між паралельними прямими \( 6x + 8y - 5 = 0 \) і \( y = -\frac{3}{4}x + \frac{15}{8} \).

Візьмемо точку на другій прямій: при \( x = 0 \), \( y = \frac{15}{8} \), тобто \( M(0; \frac{15}{8}) \).

\[ d = \frac{\left|6 \cdot 0 + 8 \cdot \frac{15}{8} - 5\right|}{\sqrt{6^2 + 8^2}} = \frac{|15 - 5|}{\sqrt{100}} = \frac{10}{10} = 1. \]

Відповідь: \( d = 1 \).

Примітка. Відстань між паралельними прямими

\[ Ax + By + C_1 = 0 \quad \text{і} \quad Ax + By + C_2 = 0 \]

обчислюється за формулою:

\[ d = \frac{|C_2 - C_1|}{\sqrt{A^2 + B^2}} \quad (9.21) \]

Для прикладу 9.20: друга пряма у загальному вигляді \( 6x + 8y - 15 = 0 \), тобто \( C_1 = -5 \), \( C_2 = -15 \).

\[ d = \frac{|-15 - (-5)|}{\sqrt{36 + 64}} = \frac{10}{10} = 1. \]

Загальне рівняння прямої та його окремі випадки

1. Яка формула відстані від точки до прямої?
\[ d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}} \]
2. Як обчислити відстань між двома паралельними прямими
\( Ax + By + C_1 = 0 \) і \( Ax + By + C_2 = 0 \)?
\[ d = \frac{|C_2 - C_1|}{\sqrt{A^2 + B^2}} \]
3. Що означає \(d=0\)?
Точка лежить на прямій
