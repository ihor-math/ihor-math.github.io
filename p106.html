3. –Ø–∫—â–æ \(y = \varphi(x)\) —ñ \(y = \psi(x)\) ‚Äì –ª—ñ–Ω—ñ–π–Ω–æ –Ω–µ–∑–∞–ª–µ–∂–Ω—ñ —Ä–æ–∑–≤‚Äô—è–∑–∫–∏ —Ä—ñ–≤–Ω—è–Ω–Ω—è (17.16), —Ç–æ –∑–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫ –º–∞—î –≤–∏–≥–ª—è–¥

–¥–µ \(C_1\) —Ç–∞ \(C_2\) ‚Äì –¥–æ–≤—ñ–ª—å–Ω—ñ —Å—Ç–∞–ª—ñ.

–ü—Ä–∏–º—ñ—Ç–∫–∞. –î–≤–∞ —Ä–æ–∑–≤‚Äô—è–∑–∫–∏ –Ω–∞–∑–∏–≤–∞—é—Ç—å –ª—ñ–Ω—ñ–π–Ω–æ –∑–∞–ª–µ–∂–Ω–∏–º–∏, —è–∫—â–æ –æ–¥–∏–Ω —ñ–∑ –Ω–∏—Ö —î –¥–æ–±—É—Ç–∫–æ–º —ñ–Ω—à–æ–≥–æ –Ω–∞ —Å—Ç–∞–ª—É, —ñ –ª—ñ–Ω—ñ–π–Ω–æ –Ω–µ–∑–∞–ª–µ–∂–Ω–∏–º–∏ ‚Äì —è–∫—â–æ –∂–æ–¥–µ–Ω —ñ–∑ –Ω–∏—Ö –Ω–µ —î –¥–æ–±—É—Ç–∫–æ–º —ñ–Ω—à–æ–≥–æ –Ω–∞ —Å—Ç–∞–ª—É.

–ú–µ—Ç–æ–¥ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏—á–Ω–æ–≥–æ —Ä—ñ–≤–Ω—è–Ω–Ω—è

–†–æ–∑–≤‚Äô—è–∑–∫–∏ —Ä—ñ–≤–Ω—è–Ω–Ω—è (17.16) —à—É–∫–∞—é—Ç—å —É –≤–∏–≥–ª—è–¥—ñ \(y = e^{\lambda x}\). –ü—ñ–¥—Å—Ç–∞–≤–ª—è—é—á–∏ —Ü–µ–π –≤–∏—Ä–∞–∑ —É —Ä—ñ–≤–Ω—è–Ω–Ω—è (17.16), –æ—Ç—Ä–∏–º—É—î–º–æ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏—á–Ω–µ —Ä—ñ–≤–Ω—è–Ω–Ω—è:

–ó–∞–ª–µ–∂–Ω–æ –≤—ñ–¥ –∫–æ—Ä–µ–Ω—ñ–≤ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏—á–Ω–æ–≥–æ —Ä—ñ–≤–Ω—è–Ω–Ω—è —Ä–æ–∑—Ä—ñ–∑–Ω—è—é—Ç—å —Ç—Ä–∏ –≤–∏–ø–∞–¥–∫–∏.

1. –í–∏–ø–∞–¥–æ–∫: –¥—ñ–π—Å–Ω—ñ —Ä—ñ–∑–Ω—ñ –∫–æ—Ä–µ–Ω—ñ

–ö–æ—Ä–µ–Ω—ñ \(\lambda_1\) —ñ \(\lambda_2\) —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏—á–Ω–æ–≥–æ —Ä—ñ–≤–Ω—è–Ω–Ω—è (17.17) ‚Äì –¥—ñ–π—Å–Ω—ñ —Ç–∞ —Ä—ñ–∑–Ω—ñ. –¢–æ–¥—ñ \(y = e^{\lambda_1 x}\) —ñ \(y = e^{\lambda_2 x}\) ‚Äì –ª—ñ–Ω—ñ–π–Ω–æ –Ω–µ–∑–∞–ª–µ–∂–Ω—ñ —Ä–æ–∑–≤‚Äô—è–∑–∫–∏, –∞ –∑–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫ –º–∞—î –≤–∏–≥–ª—è–¥:

\[ y = C_1 e^{\lambda_1 x} + C_2 e^{\lambda_2 x}. \quad (17.18) \]
–ü—Ä–∏–∫–ª–∞–¥ 17.12. –†–æ–∑–≤‚Äô—è–∑–∞—Ç–∏ —Ä—ñ–≤–Ω—è–Ω–Ω—è \(y'' - 4y' + 3y = 0\).

–†–æ–∑–≤'—è–∑–∞–Ω–Ω—è:

–°–∫–ª–∞–¥–µ–º–æ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏—á–Ω–µ —Ä—ñ–≤–Ω—è–Ω–Ω—è:

\[ \lambda^2 - 4\lambda + 3 = 0, \]

–∑–≤—ñ–¥–∫–∏ \(\lambda_1 = 3\) —ñ \(\lambda_2 = 1\).

–û—Ç–∂–µ, –ª—ñ–Ω—ñ–π–Ω–æ –Ω–µ–∑–∞–ª–µ–∂–Ω—ñ —Ä–æ–∑–≤‚Äô—è–∑–∫–∏: \(y = e^{3x}\) —ñ \(y = e^x\).

–ó–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫:

\[ y = C_1 e^{3x} + C_2 e^x. \]
–í—ñ–¥–ø–æ–≤—ñ–¥—å: \( y = C_1 e^{3x} + C_2 e^x \)

2. –í–∏–ø–∞–¥–æ–∫: –¥—ñ–π—Å–Ω—ñ –æ–¥–Ω–∞–∫–æ–≤—ñ –∫–æ—Ä–µ–Ω—ñ

–ö–æ—Ä–µ–Ω—ñ \(\lambda_1\) —ñ \(\lambda_2\) —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏—á–Ω–æ–≥–æ —Ä—ñ–≤–Ω—è–Ω–Ω—è (17.17) ‚Äì –¥—ñ–π—Å–Ω—ñ –π –æ–¥–Ω–∞–∫–æ–≤—ñ. –¢–æ–¥—ñ \(y = e^{\lambda_1 x}\) —Ç–∞ \(y = x e^{\lambda_1 x}\) ‚Äì –ª—ñ–Ω—ñ–π–Ω–æ –Ω–µ–∑–∞–ª–µ–∂–Ω—ñ —Ä–æ–∑–≤‚Äô—è–∑–∫–∏, –∞ –∑–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫ –º–∞—î –≤–∏–≥–ª—è–¥:

–ü—Ä–∏–∫–ª–∞–¥ 17.13. –†–æ–∑–≤‚Äô—è–∑–∞—Ç–∏ —Ä—ñ–≤–Ω—è–Ω–Ω—è \(y'' - 8y' + 16y = 0\).

–†–æ–∑–≤'—è–∑–∞–Ω–Ω—è:

–°–∫–ª–∞–¥–µ–º–æ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏—á–Ω–µ —Ä—ñ–≤–Ω—è–Ω–Ω—è:

–û—Ç–∂–µ, –ª—ñ–Ω—ñ–π–Ω–æ –Ω–µ–∑–∞–ª–µ–∂–Ω—ñ —Ä–æ–∑–≤‚Äô—è–∑–∫–∏: \(y = e^{4x}\), \(y = x e^{4x}\).

–ó–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫:

–í—ñ–¥–ø–æ–≤—ñ–¥—å: \( y = C_1 e^{4x} + C_2 x e^{4x} \)

3. –í–∏–ø–∞–¥–æ–∫: –∫–æ–º–ø–ª–µ–∫—Å–Ω—ñ –∫–æ—Ä–µ–Ω—ñ

–ö–æ—Ä–µ–Ω—ñ \(\lambda_1\) —ñ \(\lambda_2\) —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏—á–Ω–æ–≥–æ —Ä—ñ–≤–Ω—è–Ω–Ω—è (17.17) ‚Äì –∫–æ–º–ø–ª–µ–∫—Å–Ω—ñ: \(\lambda_1 = a + bi\) —Ç–∞ \(\lambda_2 = a - bi\) (\(b \ne 0\)), –∞ –∑–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫ –º–∞—î –≤–∏–≥–ª—è–¥:

\[ y = e^{ax} (C_1 \cos bx + C_2 \sin bx). \quad (17.20) \]
–ü—Ä–∏–∫–ª–∞–¥ 17.14. –†–æ–∑–≤‚Äô—è–∑–∞—Ç–∏ —Ä—ñ–≤–Ω—è–Ω–Ω—è \(y'' - 4y' + 13y = 0\).

–†–æ–∑–≤'—è–∑–∞–Ω–Ω—è:

–•–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏—á–Ω–µ —Ä—ñ–≤–Ω—è–Ω–Ω—è

\[ \lambda^2 - 4\lambda + 13 = 0 \]

–º–∞—î –∫–æ—Ä–µ–Ω—ñ \(\lambda_1 = 2 + 3i\) —Ç–∞ \(\lambda_2 = 2 - 3i\).

–û—Ç–∂–µ, –∑–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫:

\[ y = e^{2x} (C_1 \cos 3x + C_2 \sin 3x). \]
–í—ñ–¥–ø–æ–≤—ñ–¥—å: \( y = e^{2x} (C_1 \cos 3x + C_2 \sin 3x) \)
–ó–∞–¥–∞—á–∞ –ö–æ—à—ñ

–Ø–∫—â–æ –∑–∞–¥–∞–Ω—ñ –ø–æ—á–∞—Ç–∫–æ–≤—ñ —É–º–æ–≤–∏, —Ç–æ –º–æ–∂–Ω–∞ –∑–Ω–∞–π—Ç–∏ –Ω–µ –∑–∞–≥–∞–ª—å–Ω–∏–π, –∞ –∫–æ–Ω–∫—Ä–µ—Ç–Ω–∏–π (—á–∞—Å—Ç–∫–æ–≤–∏–π) —Ä–æ–∑–≤‚Äô—è–∑–æ–∫, —è–∫–∏–π –Ω–∞–∑–∏–≤–∞—î—Ç—å—Å—è —Ä–æ–∑–≤‚Äô—è–∑–∫–æ–º –∑–∞–¥–∞—á—ñ –ö–æ—à—ñ.

–ü—Ä–∏–∫–ª–∞–¥ 17.15. –†–æ–∑–≤‚Äô—è–∑–∞—Ç–∏ –∑–∞–¥–∞—á—É –ö–æ—à—ñ \[ \begin{cases} y'' - 3y' - 4y = 0, \\ y(0) = 6, \\ y'(0) = -1. \end{cases} \]

–†–æ–∑–≤'—è–∑–∞–Ω–Ω—è:

–•–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏—á–Ω–µ —Ä—ñ–≤–Ω—è–Ω–Ω—è

\[ \lambda^2 - 3\lambda - 4 = 0, \]

–º–∞—î –∫–æ—Ä–µ–Ω—ñ \(\lambda_1 = -1\) —ñ \(\lambda_2 = 4\).

–ó–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫:

–¥–µ \(C_1\) —Ç–∞ \(C_2\) ‚Äì –¥–æ–≤—ñ–ª—å–Ω—ñ —Å—Ç–∞–ª—ñ, —è–∫—ñ —Ç—Ä–µ–±–∞ –≤–∏–∑–Ω–∞—á–∏—Ç–∏ –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –ø–æ—á–∞—Ç–∫–æ–≤–∏—Ö —É–º–æ–≤.

\[ y(0) = C_1 e^{0} + C_2 e^{0} = C_1 + C_2 = 6. \]

\[ \begin{cases} C_1 + C_2 = 6, \\ -C_1 + 4C_2 = -1. \end{cases} \]

–î–æ–¥–∞–º–æ —Ä—ñ–≤–Ω—è–Ω–Ω—è, –æ—Ç—Ä–∏–º–∞—î–º–æ \(5C_2 = 5\), –∑–≤—ñ–¥–∫–∏ \(C_2 = 1\).

–ü—ñ–¥—Å—Ç–∞–≤–∏–º–æ \(C_2 = 1\) —É –ø–µ—Ä—à–µ —Ä—ñ–≤–Ω—è–Ω–Ω—è: \(C_1 + 1 = 6\), –æ—Ç–∂–µ \(C_1 = 5\).

–¢–∞–∫–∏–º —á–∏–Ω–æ–º, —Ä–æ–∑–≤‚Äô—è–∑–æ–∫ –∑–∞–¥–∞—á—ñ –ö–æ—à—ñ:

\[ y = 5e^{-x} + e^{4x}. \]
–í—ñ–¥–ø–æ–≤—ñ–¥—å: \( y = 5e^{-x} + e^{4x} \)

–í—ñ–¥–µ–æ–º–∞—Ç–µ—Ä—ñ–∞–ª–∏

–î–∏—Ñ–µ—Ä–µ–Ω—Ü—ñ–∞–ª—å–Ω—ñ —Ä—ñ–≤–Ω—è–Ω–Ω—è –¥—Ä—É–≥–æ–≥–æ –ø–æ—Ä—è–¥–∫—É

–î–∏—Ñ–µ—Ä–µ–Ω—Ü—ñ–∞–ª—å–Ω—ñ —Ä—ñ–≤–Ω—è–Ω–Ω—è –¥—Ä—É–≥–æ–≥–æ –ø–æ—Ä—è–¥–∫—É. –ó–∞–¥–∞—á–∞ –ö–æ—à—ñ

1. –Ø–∫–∏–π –≤–∏–≥–ª—è–¥ –º–∞—î –æ–¥–Ω–æ—Ä—ñ–¥–Ω–µ –¥–∏—Ñ–µ—Ä–µ–Ω—Ü—ñ–∞–ª—å–Ω–µ —Ä—ñ–≤–Ω—è–Ω–Ω—è –¥—Ä—É–≥–æ–≥–æ –ø–æ—Ä—è–¥–∫—É –∑ –ø–æ—Å—Ç—ñ–π–Ω–∏–º–∏ –∫–æ–µ—Ñ—ñ—Ü—ñ—î–Ω—Ç–∞–º–∏?

2. –©–æ —Ç–∞–∫–µ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏—á–Ω–µ —Ä—ñ–≤–Ω—è–Ω–Ω—è –¥–ª—è —Ä—ñ–≤–Ω—è–Ω–Ω—è \(y'' + py' + qy = 0\)?

3. –Ø–∫–∏–π –∑–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤'—è–∑–æ–∫, —è–∫—â–æ –∫–æ—Ä–µ–Ω—ñ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏—á–Ω–æ–≥–æ —Ä—ñ–≤–Ω—è–Ω–Ω—è –¥—ñ–π—Å–Ω—ñ –π —Ä—ñ–∑–Ω—ñ (\(\lambda_1 \ne \lambda_2\))?
