–ó–Ω–∞—á–µ–Ω–Ω—è —Ñ—É–Ω–∫—Ü—ñ—ó \(f(x_0)\) –Ω–∞–∑–∏–≤–∞—î—Ç—å—Å—è –º–∞–∫—Å–∏–º—É–º–æ–º —Ñ—É–Ω–∫—Ü—ñ—ó.

–ó–Ω–∞—á–µ–Ω–Ω—è \(f(x_0)\) –Ω–∞–∑–∏–≤–∞—î—Ç—å—Å—è –º—ñ–Ω—ñ–º—É–º–æ–º —Ñ—É–Ω–∫—Ü—ñ—ó.

–¢–æ—á–∫–∏ –º–∞–∫—Å–∏–º—É–º—É —Ç–∞ –º—ñ–Ω—ñ–º—É–º—É –Ω–∞–∑–∏–≤–∞—é—Ç—å —Ç–æ—á–∫–∞–º–∏ –µ–∫—Å—Ç—Ä–µ–º—É–º—É —ñ –ø–æ–∑–Ω–∞—á–∞—é—Ç—å \(x_{\text{max}}\) –∞–±–æ \(x_{\text{min}}\), –∞ –≤—ñ–¥–ø–æ–≤—ñ–¥–Ω—ñ –∑–Ω–∞—á–µ–Ω–Ω—è —Ñ—É–Ω–∫—Ü—ñ—ó ‚Äì –µ–∫—Å—Ç—Ä–µ–º—É–º–∞–º–∏ —Ñ—É–Ω–∫—Ü—ñ—ó —ñ –ø–æ–∑–Ω–∞—á–∞—é—Ç—å \(y_{\text{max}} = f(x_{\text{max}})\) –∞–±–æ \(y_{\text{min}} = f(x_{\text{min}})\). –í–æ–Ω–∏ —î –ª–æ–∫–∞–ª—å–Ω–∏–º–∏ –µ–∫—Å—Ç—Ä–µ–º—É–º–∞–º–∏, —Ç–æ–±—Ç–æ –Ω–∞–π–±—ñ–ª—å—à–∏–º–∏ —á–∏ –Ω–∞–π–º–µ–Ω—à–∏–º–∏ –ª–∏—à–µ –≤ –æ–∫–æ–ª—ñ —Ç–æ—á–∫–∏, –∞ –Ω–µ –Ω–∞ –≤—Å—ñ–π –æ–±–ª–∞—Å—Ç—ñ –≤–∏–∑–Ω–∞—á–µ–Ω–Ω—è —Ñ—É–Ω–∫—Ü—ñ—ó. –§—É–Ω–∫—Ü—ñ—è \(y=f(x)\) –º–æ–∂–µ –º–∞—Ç–∏ –∫—ñ–ª—å–∫–∞ —Ç–∞–∫–∏—Ö —Ç–æ—á–æ–∫.

–ó–≤'—è–∑–æ–∫ —ñ–∑ –ø–æ—Ö—ñ–¥–Ω–æ—é. –ù–µ–æ–±—Ö—ñ–¥–Ω–∞ —É–º–æ–≤–∞ —ñ—Å–Ω—É–≤–∞–Ω–Ω—è –µ–∫—Å—Ç—Ä–µ–º—É–º—É

–Ø–∫—â–æ —Ñ—É–Ω–∫—Ü—ñ—è \(y=f(x)\) –º–∞—î –µ–∫—Å—Ç—Ä–µ–º—É–º –≤ —Ç–æ—á—Ü—ñ \(x_0\), —Ç–æ —ó—ó –ø–æ—Ö—ñ–¥–Ω–∞ \(f'(x_0) = 0\) –∞–±–æ \(f'(x_0)\) –Ω–µ —ñ—Å–Ω—É—î. –¢–∞–∫—É —Ç–æ—á–∫—É \(x_0\) –Ω–∞–∑–∏–≤–∞—é—Ç—å –∫—Ä–∏—Ç–∏—á–Ω–æ—é —Ç–æ—á–∫–æ—é.

–ó–≤–µ—Ä–Ω—É—Ç–∏ —É–≤–∞–≥—É:

—Ü—ñ —É–º–æ–≤–∏ –Ω–µ —î –¥–æ—Å—Ç–∞—Ç–Ω—ñ–º–∏ –¥–ª—è —ñ—Å–Ω—É–≤–∞–Ω–Ω—è –µ–∫—Å—Ç—Ä–µ–º—É–º—É. –§—É–Ω–∫—Ü—ñ—è –º–æ–∂–µ –º–∞—Ç–∏ –Ω—É–ª—å–æ–≤—É –ø–æ—Ö—ñ–¥–Ω—É –≤ —Ç–æ—á—Ü—ñ, –∞–ª–µ —Ü—è —Ç–æ—á–∫–∞ –Ω–µ –æ–±–æ–≤'—è–∑–∫–æ–≤–æ —î —Ç–æ—á–∫–æ—é –µ–∫—Å—Ç—Ä–µ–º—É–º—É. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, —É —Ñ—É–Ω–∫—Ü—ñ—ó \(y=x^3\) –≤ —Ç–æ—á—Ü—ñ \((0;0)\) –ø–æ—Ö—ñ–¥–Ω–∞ –¥–æ—Ä—ñ–≤–Ω—é—î –Ω—É–ª—é, –∞–ª–µ –µ–∫—Å—Ç—Ä–µ–º—É–º—É –Ω–µ–º–∞—î ‚Äì —Ü–µ —Ç–æ—á–∫–∞ –ø–µ—Ä–µ–≥–∏–Ω—É (—Ä–∏—Å. 14.7).

–©–æ–± –ø–µ—Ä–µ–≤—ñ—Ä–∏—Ç–∏, —á–∏ –º–∞—î —Ñ—É–Ω–∫—Ü—ñ—è –≤ –∫—Ä–∏—Ç–∏—á–Ω—ñ–π —Ç–æ—á—Ü—ñ –µ–∫—Å—Ç—Ä–µ–º—É–º, –Ω–µ–æ–±—Ö—ñ–¥–Ω–æ –¥–æ–¥–∞—Ç–∫–æ–≤–µ –¥–æ—Å–ª—ñ–¥–∂–µ–Ω–Ω—è. –î–ª—è —Ü—å–æ–≥–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å –¥–æ—Å—Ç–∞—Ç–Ω—ñ —É–º–æ–≤–∏ —ñ—Å–Ω—É–≤–∞–Ω–Ω—è –µ–∫—Å—Ç—Ä–µ–º—É–º—É.

–î–æ—Å—Ç–∞—Ç–Ω—ñ —É–º–æ–≤–∏ —ñ—Å–Ω—É–≤–∞–Ω–Ω—è –µ–∫—Å—Ç—Ä–µ–º—É–º—É

–ü–µ—Ä—à–∞ –¥–æ—Å—Ç–∞—Ç–Ω—è —É–º–æ–≤–∞

–Ø–∫—â–æ –Ω–µ–ø–µ—Ä–µ—Ä–≤–Ω–∞ —Ñ—É–Ω–∫—Ü—ñ—è \(y=f(x)\) –º–∞—î –ø–æ—Ö—ñ–¥–Ω—É —É –≤—Å—ñ—Ö —Ç–æ—á–∫–∞—Ö —ñ–Ω—Ç–µ—Ä–≤–∞–ª—É, —â–æ –º—ñ—Å—Ç–∏—Ç—å –∫—Ä–∏—Ç–∏—á–Ω—É —Ç–æ—á–∫—É \(x_0\), (–∫—Ä—ñ–º, –º–æ–∂–ª–∏–≤–æ, —Å–∞–º–æ—ó \(x_0\)) —ñ –ø–æ—Ö—ñ–¥–Ω–∞ \(f'(x)\) –∑–º—ñ–Ω—é—î –∑–Ω–∞–∫ –∑ ¬´–ø–ª—é—Å–∞ –Ω–∞ –º—ñ–Ω—É—Å¬ª –ø—Ä–∏ –ø–µ—Ä–µ—Ö–æ–¥—ñ —á–µ—Ä–µ–∑ —Ç–æ—á–∫—É \(x_0\), —Ç–æ —Ñ—É–Ω–∫—Ü—ñ—è –≤ —Ü—ñ–π —Ç–æ—á—Ü—ñ –º–∞—î –º–∞–∫—Å–∏–º—É–º (—Ä–∏—Å. 14.8).

–Ø–∫—â–æ –∂ –ø–æ—Ö—ñ–¥–Ω–∞ –∑–º—ñ–Ω—é—î –∑–Ω–∞–∫ –∑ ¬´–º—ñ–Ω—É—Å–∞ –Ω–∞ –ø–ª—é—Å¬ª, —Ç–æ —Ñ—É–Ω–∫—Ü—ñ—è –º–∞—î –º—ñ–Ω—ñ–º—É–º (—Ä–∏—Å. 14.9).

–î—Ä—É–≥–∞ –¥–æ—Å—Ç–∞—Ç–Ω—è —É–º–æ–≤–∞

–Ø–∫—â–æ –≤ —Ç–æ—á—Ü—ñ \(x_0\) –ø–µ—Ä—à–∞ –ø–æ—Ö—ñ–¥–Ω–∞ \(f'(x_0)=0\), –∞ –¥—Ä—É–≥–∞ –ø–æ—Ö—ñ–¥–Ω–∞ \(f''(x_0) \neq 0\), —Ç–æ –ø—Ä–∏ \(f''(x_0) < 0\) —Ñ—É–Ω–∫—Ü—ñ—è –º–∞—î –º–∞–∫—Å–∏–º—É–º, –∞ –ø—Ä–∏ \(f''(x_0) > 0\) ‚Äì –º—ñ–Ω—ñ–º—É–º.

\[ \begin{cases} f'(x_0) = 0, \ f''(x_0) < 0 & \Rightarrow \text{–º–∞–∫—Å–∏–º—É–º} \\ f'(x_0) = 0, \ f''(x_0) > 0 & \Rightarrow \text{–º—ñ–Ω—ñ–º—É–º} \\ f'(x_0) = 0, \ f''(x_0) = 0 & \Rightarrow \text{–Ω–µ –∑–∞—Å—Ç–æ—Å–æ–≤—É—î—Ç—å—Å—è} \end{cases} \]

üìù –ó–∞—É–≤–∞–∂–µ–Ω–Ω—è:

1. –Ø–∫—â–æ \(x_0\) ‚Äì —Ç–æ—á–∫–∞ —Ä–æ–∑—Ä–∏–≤—É —Ñ—É–Ω–∫—Ü—ñ—ó, –≤–æ–Ω–∞ –Ω–µ –º–æ–∂–µ –±—É—Ç–∏ —Ç–æ—á–∫–æ—é –µ–∫—Å—Ç—Ä–µ–º—É–º—É, –Ω–∞–≤—ñ—Ç—å —è–∫—â–æ –ø–æ—Ö—ñ–¥–Ω–∞ –∑–º—ñ–Ω—é—î –∑–Ω–∞–∫ –ø—Ä–∏ –ø–µ—Ä–µ—Ö–æ–¥—ñ —á–µ—Ä–µ–∑ –Ω–µ—ó, —á–µ—Ä–µ–∑ –ø–æ—Ä—É—à–µ–Ω–Ω—è –Ω–µ–ø–µ—Ä–µ—Ä–≤–Ω–æ—Å—Ç—ñ.

2. –Ø–∫—â–æ –≤ –∫—Ä–∏—Ç–∏—á–Ω—ñ–π —Ç–æ—á—Ü—ñ \(x_0\) –¥—Ä—É–≥–∞ –ø–æ—Ö—ñ–¥–Ω–∞ \(f''(x_0)=0\) –∞–±–æ \(f''(x_0)\) –Ω–µ —ñ—Å–Ω—É—î, —Ç–æ –¥—Ä—É–≥–∞ –¥–æ—Å—Ç–∞—Ç–Ω—è —É–º–æ–≤–∞ –Ω–µ –∑–∞—Å—Ç–æ—Å–æ–≤—É—î—Ç—å—Å—è.

–ê–ª–≥–æ—Ä–∏—Ç–º –∑–Ω–∞—Ö–æ–¥–∂–µ–Ω–Ω—è –µ–∫—Å—Ç—Ä–µ–º—É–º—ñ–≤ —Ñ—É–Ω–∫—Ü—ñ—ó

  1. –ó–Ω–∞–π—Ç–∏ –æ–±–ª–∞—Å—Ç—å –≤–∏–∑–Ω–∞—á–µ–Ω–Ω—è —Ñ—É–Ω–∫—Ü—ñ—ó \(D(f)\).
  2. –ó–Ω–∞–π—Ç–∏ –ø–æ—Ö—ñ–¥–Ω—É \(f'(x)\) —Ñ—É–Ω–∫—Ü—ñ—ó.
  3. –ó–Ω–∞–π—Ç–∏ –∫—Ä–∏—Ç–∏—á–Ω—ñ —Ç–æ—á–∫–∏:
    • –†–æ–∑–≤'—è–∑–∞—Ç–∏ —Ä—ñ–≤–Ω—è–Ω–Ω—è \(f'(x) = 0\) (—Å—Ç–∞—Ü—ñ–æ–Ω–∞—Ä–Ω—ñ —Ç–æ—á–∫–∏)
    • –ó–Ω–∞–π—Ç–∏ —Ç–æ—á–∫–∏, –¥–µ –ø–æ—Ö—ñ–¥–Ω–∞ \(f'(x)\) –Ω–µ —ñ—Å–Ω—É—î
  4. –í—ñ–¥—ñ–±—Ä–∞—Ç–∏ –∫—Ä–∏—Ç–∏—á–Ω—ñ —Ç–æ—á–∫–∏, —â–æ –Ω–∞–ª–µ–∂–∞—Ç—å –æ–±–ª–∞—Å—Ç—ñ –≤–∏–∑–Ω–∞—á–µ–Ω–Ω—è \(D(f)\).
  5. –î–æ—Å–ª—ñ–¥–∏—Ç–∏ –∑–Ω–∞–∫ –ø–æ—Ö—ñ–¥–Ω–æ—ó \(f'(x)\) –Ω–∞ —ñ–Ω—Ç–µ—Ä–≤–∞–ª–∞—Ö, –Ω–∞ —è–∫—ñ –∫—Ä–∏—Ç–∏—á–Ω—ñ —Ç–æ—á–∫–∏ —Ä–æ–∑–±–∏–≤–∞—é—Ç—å –æ–±–ª–∞—Å—Ç—å –≤–∏–∑–Ω–∞—á–µ–Ω–Ω—è.
  6. –ó–∞ –ø–µ—Ä—à–æ—é –¥–æ—Å—Ç–∞—Ç–Ω—å–æ—é —É–º–æ–≤–æ—é –≤–∏–∑–Ω–∞—á–∏—Ç–∏ —Ö–∞—Ä–∞–∫—Ç–µ—Ä –∫–æ–∂–Ω–æ—ó –∫—Ä–∏—Ç–∏—á–Ω–æ—ó —Ç–æ—á–∫–∏:
    • –Ø–∫—â–æ –ø–æ—Ö—ñ–¥–Ω–∞ –∑–º—ñ–Ω—é—î –∑–Ω–∞–∫ –∑ "+" –Ω–∞ "‚àí" ‚Üí —Ç–æ—á–∫–∞ –º–∞–∫—Å–∏–º—É–º—É
    • –Ø–∫—â–æ –ø–æ—Ö—ñ–¥–Ω–∞ –∑–º—ñ–Ω—é—î –∑–Ω–∞–∫ –∑ "‚àí" –Ω–∞ "+" ‚Üí —Ç–æ—á–∫–∞ –º—ñ–Ω—ñ–º—É–º—É
    • –Ø–∫—â–æ –ø–æ—Ö—ñ–¥–Ω–∞ –Ω–µ –∑–º—ñ–Ω—é—î –∑–Ω–∞–∫ ‚Üí –µ–∫—Å—Ç—Ä–µ–º—É–º—É –Ω–µ–º–∞—î
  7. –û–±—á–∏—Å–ª–∏—Ç–∏ –∑–Ω–∞—á–µ–Ω–Ω—è —Ñ—É–Ω–∫—Ü—ñ—ó \(f(x)\) –≤ —Ç–æ—á–∫–∞—Ö –µ–∫—Å—Ç—Ä–µ–º—É–º—É.
  8. –ó–∞–ø–∏—Å–∞—Ç–∏ –≤—ñ–¥–ø–æ–≤—ñ–¥—å: –∫–æ–æ—Ä–¥–∏–Ω–∞—Ç–∏ —Ç–æ—á–æ–∫ –µ–∫—Å—Ç—Ä–µ–º—É–º—É —Ç–∞ –∑–Ω–∞—á–µ–Ω–Ω—è –µ–∫—Å—Ç—Ä–µ–º—É–º—ñ–≤.

* –î–ª—è –ø–µ—Ä–µ–≤—ñ—Ä–∫–∏ –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞—Ç–∏ –¥—Ä—É–≥—É –¥–æ—Å—Ç–∞—Ç–Ω—é —É–º–æ–≤—É, —è–∫—â–æ –≤–æ–Ω–∞ –∑–∞—Å—Ç–æ—Å–æ–≤–Ω–∞.

–ü—Ä–∏–∫–ª–∞–¥–∏ –∑–Ω–∞—Ö–æ–¥–∂–µ–Ω–Ω—è –µ–∫—Å—Ç—Ä–µ–º—É–º—É
–ü—Ä–∏–∫–ª–∞–¥ 14.2. –í–∏–∑–Ω–∞—á–∏—Ç–∏ –ø—Ä–æ–º—ñ–∂–∫–∏ –º–æ–Ω–æ—Ç–æ–Ω–Ω–æ—Å—Ç—ñ —Ç–∞ —Ç–æ—á–∫–∏ –µ–∫—Å—Ç—Ä–µ–º—É–º—É —Ñ—É–Ω–∫—Ü—ñ—ó \( y = 2x + 3\sqrt[3]{x^2} \).

‚ñ† –û–±–ª–∞—Å—Ç—å –≤–∏–∑–Ω–∞—á–µ–Ω–Ω—è —Ñ—É–Ω–∫—Ü—ñ—ó: \(x \in (-\infty; +\infty)\). –ó–Ω–∞–π–¥–µ–º–æ –ø–æ—Ö—ñ–¥–Ω—É —Ñ—É–Ω–∫—Ü—ñ—ó:

\[ y' = (2x + 3x^{2/3})' = 2 + 3 \cdot \frac{2}{3} x^{-1/3} = 2 + \frac{2}{\sqrt[3]{x}}, \]
\[ y' = \frac{2\sqrt[3]{x} + 2}{\sqrt[3]{x}}. \]

–ó–Ω–∞–π–¥–µ–º–æ –∫—Ä–∏—Ç–∏—á–Ω—ñ —Ç–æ—á–∫–∏. –°–ø–æ—á–∞—Ç–∫—É —Ç–æ—á–∫–∏ —É —è–∫–∏—Ö –ø–æ—Ö—ñ–¥–Ω–∞ –¥–æ—Ä—ñ–≤–Ω—é—î –Ω—É–ª—é (—Å—Ç–∞—Ü—ñ–æ–Ω–∞—Ä–Ω—ñ —Ç–æ—á–∫–∏):

\[ y' = 0 \implies 2\sqrt[3]{x} + 2 = 0, \]
\[ \sqrt[3]{x} = -1 \implies x = -1. \]
