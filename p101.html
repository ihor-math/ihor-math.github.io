–í–∞–∂–ª–∏–≤–æ! –î–ª—è —É—Å–ø—ñ—à–Ω–æ–≥–æ –≤–∏–≤—á–µ–Ω–Ω—è –¥–∏—Ñ–µ—Ä–µ–Ω—Ü—ñ–∞–ª—å–Ω–∏—Ö —Ä—ñ–≤–Ω—è–Ω—å –Ω–µ–æ–±—Ö—ñ–¥–Ω–æ –¥–æ–±—Ä–µ –∑–Ω–∞—Ç–∏ –º–µ—Ç–æ–¥–∏ –¥–∏—Ñ–µ—Ä–µ–Ω—Ü—ñ—é–≤–∞–Ω–Ω—è —Ç–∞ —ñ–Ω—Ç–µ–≥—Ä—É–≤–∞–Ω–Ω—è.

–ú–µ—Ç–æ–¥–∏ —Ä–æ–∑–≤‚Äô—è–∑–∞–Ω–Ω—è –Ω–∞–π–ø—Ä–æ—Å—Ç—ñ—à–∏—Ö –¥–∏—Ñ–µ—Ä–µ–Ω—Ü—ñ–∞–ª—å–Ω–∏—Ö —Ä—ñ–≤–Ω—è–Ω—å

1. –Ü–Ω—Ç–µ–≥—Ä—É–≤–∞–Ω–Ω—è –æ–±–æ—Ö —á–∞—Å—Ç–∏–Ω —Ä—ñ–≤–Ω—è–Ω–Ω—è

–©–æ–± –∑–Ω–∞–π—Ç–∏ –∑–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫ –¥–∏—Ñ–µ—Ä–µ–Ω—Ü—ñ–∞–ª—å–Ω–æ–≥–æ —Ä—ñ–≤–Ω—è–Ω–Ω—è

–Ω–∞ –∑–∞–¥–∞–Ω–æ–º—É —ñ–Ω—Ç–µ—Ä–≤–∞–ª—ñ \(X\), –ø–æ—Ç—Ä—ñ–±–Ω–æ –ø—Ä–æ—ñ–Ω—Ç–µ–≥—Ä—É–≤–∞—Ç–∏ –æ–±–∏–¥–≤—ñ —á–∞—Å—Ç–∏–Ω–∏ —Ä—ñ–≤–Ω—è–Ω–Ω—è:

\[ \int y' \,dx = \int f(x) \,dx \]

–Ü–Ω—Ç–µ–≥—Ä—É—é—á–∏, –æ—Ç—Ä–∏–º—É—î–º–æ –∑–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫:

\[ y = F(x) + C \quad (17.2) \]

–¥–µ \(F(x)\) ‚Äì –ø–µ—Ä–≤—ñ—Å–Ω–∞ —Ñ—É–Ω–∫—Ü—ñ—ó \(f(x)\) –Ω–∞ —ñ–Ω—Ç–µ—Ä–≤–∞–ª—ñ \(X\); \(C\) ‚Äì –¥–æ–≤—ñ–ª—å–Ω–∞ —Å—Ç–∞–ª–∞.

–ü—Ä–∏–∫–ª–∞–¥ 17.1. –†–æ–∑–≤‚Äô—è–∑–∞—Ç–∏ –¥–∏—Ñ–µ—Ä–µ–Ω—Ü—ñ–∞–ª—å–Ω–µ —Ä—ñ–≤–Ω—è–Ω–Ω—è \(y' = 3x^2 + 2\).

–†–æ–∑–≤'—è–∑–∞–Ω–Ω—è:

–®—É–∫–∞—î–º–æ —Ñ—É–Ω–∫—Ü—ñ—é \(y(x)\), —á–∏—è –ø–æ—Ö—ñ–¥–Ω–∞ –¥–æ—Ä—ñ–≤–Ω—é—î \(3x^2 + 2\). –î–ª—è —Ü—å–æ–≥–æ –ø—Ä–æ—ñ–Ω—Ç–µ–≥—Ä—É—î–º–æ –æ–±–∏–¥–≤—ñ —á–∞—Å—Ç–∏–Ω–∏ —Ä—ñ–≤–Ω—è–Ω–Ω—è:

\[ \int y' \,dx = \int (3x^2 + 2) \,dx \] \[ y = \int (3x^2 + 2) \,dx \]

–û—Ç–∂–µ,

\[ y = \frac{3x^3}{3} + 2x = x^3 + 2x + C \]

–¥–µ \(C\) ‚Äì –¥–æ–≤—ñ–ª—å–Ω–∞ —Å—Ç–∞–ª–∞.

–í—ñ–¥–ø–æ–≤—ñ–¥—å: \( y = x^3 + 2x + C \)

–ß–∞—Å—Ç–∫–æ–≤–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫ –¥–∏—Ñ–µ—Ä–µ–Ω—Ü—ñ–∞–ª—å–Ω–æ–≥–æ —Ä—ñ–≤–Ω—è–Ω–Ω—è

–†–æ–∑–≤‚Äô—è–∑–æ–∫ –¥–∏—Ñ–µ—Ä–µ–Ω—Ü—ñ–∞–ª—å–Ω–æ–≥–æ —Ä—ñ–≤–Ω—è–Ω–Ω—è, —è–∫–∏–π –∑–∞–¥–æ–≤–æ–ª—å–Ω—è—î –ø–æ—á–∞—Ç–∫–æ–≤—ñ —É–º–æ–≤–∏, –Ω–∞–∑–∏–≤–∞—é—Ç—å —á–∞—Å—Ç–∫–æ–≤–∏–º —Ä–æ–∑–≤‚Äô—è–∑–∫–æ–º. –ù–µ—Ö–∞–π –∑–∞–¥–∞–Ω–∞ –ø–æ—á–∞—Ç–∫–æ–≤–∞ —É–º–æ–≤–∞

\[ y(x_0) = y_0 \quad (17.3) \]

–¢–æ–¥—ñ, –∑–Ω–∞–π—à–æ–≤—à–∏ –∑–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫ \(y = F(x) + C\), –≤–∏–∑–Ω–∞—á–∞—î–º–æ —Å—Ç–∞–ª—É \(C\) –∑ –ø–æ—á–∞—Ç–∫–æ–≤–æ—ó —É–º–æ–≤–∏:

\[ F(x_0) + C = y_0 \Rightarrow C = y_0 - F(x_0) = C_0 \]

–û—Ç–∂–µ, —á–∞—Å—Ç–∫–æ–≤–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫ –º–∞—î –≤–∏–≥–ª—è–¥:

\[ y = F(x) + C_0 \quad (17.4) \]

–ó–∞–¥–∞—á—É –∑ –ø–æ—á–∞—Ç–∫–æ–≤–∏–º–∏ —É–º–æ–≤–∞–º–∏ –Ω–∞–∑–∏–≤–∞—é—Ç—å –∑–∞–¥–∞—á–µ—é –ö–æ—à—ñ.

–ü—Ä–∏–∫–ª–∞–¥ 17.2. –ó–Ω–∞–π—Ç–∏ –∑–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫ –¥–∏—Ñ–µ—Ä–µ–Ω—Ü—ñ–∞–ª—å–Ω–æ–≥–æ —Ä—ñ–≤–Ω—è–Ω–Ω—è \(y' = x\sin x\) —Ç–∞ –ø–µ—Ä–µ–≤—ñ—Ä–∏—Ç–∏ –ø—Ä–∞–≤–∏–ª—å–Ω—ñ—Å—Ç—å —Ä–µ–∑—É–ª—å—Ç–∞—Ç—É. –ó–Ω–∞–π—Ç–∏ —á–∞—Å—Ç–∫–æ–≤–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫, —â–æ –∑–∞–¥–æ–≤–æ–ª—å–Ω—è—î –ø–æ—á–∞—Ç–∫–æ–≤—É —É–º–æ–≤—É \(y(\pi/2) = 2\).

–†–æ–∑–≤'—è–∑–∞–Ω–Ω—è:

–©–æ–± –∑–Ω–∞–π—Ç–∏ –∑–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫, —ñ–Ω—Ç–µ–≥—Ä—É—î–º–æ –æ–±–∏–¥–≤—ñ —á–∞—Å—Ç–∏–Ω–∏ —Ä—ñ–≤–Ω—è–Ω–Ω—è:

\[ \int y' \,dx = \int x\sin x \,dx \] \[ y = \int x\sin x \,dx \]

–û–±—á–∏—Å–ª–∏–º–æ –π–æ–≥–æ –º–µ—Ç–æ–¥–æ–º —ñ–Ω—Ç–µ–≥—Ä—É–≤–∞–Ω–Ω—è —á–∞—Å—Ç–∏–Ω–∞–º–∏:

\[ \int x\sin x \,dx = \begin{vmatrix} u = x, & dv = \sin x \,dx \\ du = dx, & v = -\cos x \end{vmatrix} = -x\cos x + \int \cos x \,dx \] \[ = -x\cos x + \sin x + C \]

–¢–æ–∂ –∑–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫:

\[ y = -x\cos x + \sin x + C \]

–ü–µ—Ä–µ–≤—ñ—Ä–∏–º–æ –ø—Ä–∞–≤–∏–ª—å–Ω—ñ—Å—Ç—å —Ä–µ–∑—É–ª—å—Ç–∞—Ç—É. –û–±—á–∏—Å–ª–∏–º–æ –ø–æ—Ö—ñ–¥–Ω—É:

\[ y' = (-x\cos x)' + (\sin x)' + C' = -\cos x + x\sin x + \cos x = x\sin x \]

–û—Ç–∂–µ, –∑–Ω–∞–π–¥–µ–Ω–∏–π –∑–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫ –∑–∞–¥–æ–≤–æ–ª—å–Ω—è—î –¥–∞–Ω–µ –¥–∏—Ñ–µ—Ä–µ–Ω—Ü—ñ–∞–ª—å–Ω–µ —Ä—ñ–≤–Ω—è–Ω–Ω—è.

–ó–Ω–∞–π–¥–µ–º–æ —á–∞—Å—Ç–∫–æ–≤–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫. –î–ª—è —É–º–æ–≤–∏ \(y(\pi/2) = 2\) –ø—ñ–¥—Å—Ç–∞–≤–∏–º–æ \(x = \pi/2\) —ñ \(y = 2\) —É –∑–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫:

\[ 2 = -\frac{\pi}{2} \cos \frac{\pi}{2} + \sin \frac{\pi}{2} + C \]

–û—Å–∫—ñ–ª—å–∫–∏ \(\cos(\pi/2) = 0\) —ñ \(\sin(\pi/2) = 1\), —Ç–æ –º–∞—î–º–æ:

\[ 2 = 0 + 1 + C \Rightarrow C = 1 \]

–û—Ç–∂–µ, —á–∞—Å—Ç–∫–æ–≤–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫:

\[ y = -x\cos x + \sin x + 1 \]
–í—ñ–¥–ø–æ–≤—ñ–¥—å: –∑–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫: \( y = -x\cos x + \sin x + C \); —á–∞—Å—Ç–∫–æ–≤–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫: \( y = -x\cos x + \sin x + 1 \)

2. –ó–≤–∏—á–∞–π–Ω–µ –¥–∏—Ñ–µ—Ä–µ–Ω—Ü—ñ–∞–ª—å–Ω–µ —Ä—ñ–≤–Ω—è–Ω–Ω—è

\[ f(x) \cdot y' = g(x) \quad (17.5) \]

–¶–µ —Ä—ñ–≤–Ω—è–Ω–Ω—è –º–æ–∂–Ω–∞ –≤–∏—Ä–∞–∑–∏—Ç–∏ —á–µ—Ä–µ–∑ –ø–æ—Ö—ñ–¥–Ω—É, —Ä–æ–∑–¥—ñ–ª–∏–≤—à–∏ –æ–±–∏–¥–≤—ñ —á–∞—Å—Ç–∏–Ω–∏ –Ω–∞ \(f(x)\), –∑–∞ —É–º–æ–≤–∏, —â–æ \(f(x) \neq 0\) –¥–ª—è –≤—Å—ñ—Ö \(x\) –∑ –æ–±–ª–∞—Å—Ç—ñ –≤–∏–∑–Ω–∞—á–µ–Ω–Ω—è \(X\):

\[ y' = \frac{g(x)}{f(x)} \quad (17.6) \]

–©–æ–± –∑–Ω–∞–π—Ç–∏ –∑–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫, —ñ–Ω—Ç–µ–≥—Ä—É—î–º–æ –æ–±–∏–¥–≤—ñ —á–∞—Å—Ç–∏–Ω–∏ —Ä—ñ–≤–Ω—è–Ω–Ω—è:

\[ \int y' \,dx = \int \frac{g(x)}{f(x)} \,dx \]

–û—Ç–∂–µ, –∑–∞–≥–∞–ª—å–Ω–∏–π —Ä–æ–∑–≤‚Äô—è–∑–æ–∫ –º–∞—î –≤–∏–≥–ª—è–¥

\[ y = \int \frac{g(x)}{f(x)} \,dx + C \quad (17.7) \]

–¥–µ \(C\) ‚Äì –¥–æ–≤—ñ–ª—å–Ω–∞ —Å—Ç–∞–ª–∞.

–í–∞–∂–ª–∏–≤—ñ –∑–∞—É–≤–∞–∂–µ–Ω–Ω—è

1. –Ø–∫—â–æ –¥–ª—è –¥–µ—è–∫–∏—Ö \(x \in X\) –æ–¥–Ω–æ—á–∞—Å–Ω–æ —Ñ—É–Ω–∫—Ü—ñ—ó \(f(x) = 0\) —Ç–∞ \(g(x) = 0\), —Ç–æ —Ä—ñ–≤–Ω—è–Ω–Ω—è \(f(x) \cdot y' = g(x)\) –ø–µ—Ä–µ—Ç–≤–æ—Ä—é—î—Ç—å—Å—è –Ω–∞ \(0 \cdot y' = 0\) —ñ –≤–∏–∫–æ–Ω—É—î—Ç—å—Å—è –¥–ª—è –±—É–¥—å-—è–∫–æ—ó –¥–∏—Ñ–µ—Ä–µ–Ω—Ü—ñ–π–æ–≤–Ω–æ—ó —Ñ—É–Ω–∫—Ü—ñ—ó \(y(x)\).

2. –Ø–∫—â–æ –¥–ª—è –¥–µ—è–∫–∏—Ö –∑–Ω–∞—á–µ–Ω—å \(x \in X\) –≤–∏–∫–æ–Ω—É—é—Ç—å—Å—è —É–º–æ–≤–∏ \(\begin{cases} f(x) = 0 \\ g(x) \neq 0 \end{cases}\), —Ç–æ —Ä—ñ–≤–Ω—è–Ω–Ω—è \(f(x) \cdot y' = g(x)\) –Ω–µ –º–∞—î —Ä–æ–∑–≤‚Äô—è–∑–∫—ñ–≤, –æ—Å–∫—ñ–ª—å–∫–∏ –ª—ñ–≤–∞ —á–∞—Å—Ç–∏–Ω–∞ –¥–æ—Ä—ñ–≤–Ω—é—î –Ω—É–ª—é, –∞ –ø—Ä–∞–≤–∞ ‚Äì –Ω–µ–Ω—É–ª—å–æ–≤–æ–º—É –∑–Ω–∞—á–µ–Ω–Ω—é.
